## 🚀 كيفية الاستخدام

### تشغيل الموقع
```bash
cd /vercel/share/v0-project
pnpm dev
# ثم زور http://localhost:3000
```

### تغيير المحتوى
جميع النصوص والمعلومات موجودة في الملفات التالية:
- `/components/hero.tsx` - الصفحة الرئيسية
- `/components/about.tsx` - عني
- `/components/skills.tsx` - المهارات
- `/components/services.tsx` - الخدمات
- `/components/projects.tsx` - المشاريع
- وهكذا...

### تغيير الألوان
في ملف `/app/globals.css` ابحث عن:
```css
--primary: #8b5cf6;      /* البنفسجي الرئيسي */
--secondary: #a855f7;    /* البنفسجي الثانوي */
--accent: #c084fc;       /* البنفسجي الفاتح */
```
