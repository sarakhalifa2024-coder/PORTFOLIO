## 🎯 How to Customize

### Update Personal Info
Edit these files:
- `/components/hero.tsx` - Greeting & headline
- `/components/about.tsx` - Bio & summary
- `/components/skills.tsx` - Your skills
- `/components/education.tsx` - Your education
- `/components/experience.tsx` - Your experience
- `/components/projects.tsx` - Your projects
- `/components/contact.tsx` - Your contact info

### Change Colors
Edit `/app/globals.css` and update CSS variables:
```css
:root {
  --primary: #8b5cf6;        /* Your primary color */
  --secondary: #a855f7;      /* Your secondary color */
  --accent: #c084fc;         /* Your accent color */
}
```

### Replace Images
Put your images in `/public/` and update component imports.

### Update Pricing
Edit `/components/pricing.tsx` and modify the `plans` array.

### Change Email
Update in `/components/contact.tsx`:
```tsx
<a href="mailto:your@email.com">
```

## 🐛 Troubleshooting

### Images not showing?
Check `/public/` folder and image paths in components.

### Dark mode not working?
Clear `.next` folder: `rm -rf .next`

### Animations not smooth?
Check browser supports CSS animations (all modern browsers do).

### Colors not changing?
Update CSS variables in `/app/globals.css`

### Theme not persisting?
Already handled by `next-themes` with localStorage.
